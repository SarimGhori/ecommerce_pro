Mini E-Commerce Website (PHP)

This is a simple Mini E-Commerce Website made using PHP and MySQL.

Features

Show products

Product details page

Add to cart (simple)

Checkout page (basic)

Files

index.php – home page

products.php – all products

product.php – single product

cart.php – add/remove items

config.php – database connection

How to Run

Put the project in xampp/htdocs

Start Apache and MySQL in XAMPP

Import the SQL file into phpMyAdmin

Open in browser:
